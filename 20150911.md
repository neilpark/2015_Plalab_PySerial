# XML Parsing
```python
from xml.etree.ElementTree import parse
class XmlFile:
    def __init__(self, path):
        self.path = path
        self.tree = parse(path)
        self.root_tag = self.tree.getroot()
        self.find_tag_info(self.root_tag)
```
